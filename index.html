I wrote a web scraper in Python that inserts the text of Wall Street Journal articles into a SQL database (https://github.com/spwg/wsj-scraper). Right now it navigates to the WSJ login page, logins in using my credentials, clicks the top link on the homepage, logs that article’s text, and then moves onto the first link the article’s content. My friend had a meeting with the sentiment trading group of Brown Trading Club, and I overheard them talking about this project. That's how I got involved. (October 2017.)
